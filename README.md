# Тестовое задание: Компьютерный клуб на C++

## О программе
Программа симулирует работу компьютерного клуба, обрабатывая события в реальном времени:
- Приход и уход клиентов
- Посадка за столы и смена столов
- Очередь ожидания
- Расчёт выручки с почасовой оплатой
- Генерация событий

Программа покрыта юнит-тестами, которые проверяют основной функционал и логику программы. В качестве фреймворка для тестирования был выбрал [doctest](https://github.com/doctest/doctest). С тестами можно ознакомиться ниже. 
# Сборка и запуск:

## Linux
```bash
#Основная программа
g++ main.cpp Parser.cpp Club.cpp -o task
./task test.txt

#Тесты
g++ tests.cpp Parser.cpp Club.cpp -o tests
./tests
```

## Windows
```shell
#Основная программа
g++ main.cpp Parser.cpp Club.cpp -o task.exe
task.exe test.txt

#Тесты
g++ tests.cpp Parser.cpp Club.cpp -o tests.exe
tests.exe
```

Результатом программы является файл `out_task.txt`.
Результатом тестов являются соответствующие записи в терминале о прохождении тестов.

_Программа написана в качестве тестового задания._
